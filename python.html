What is Python?

Python was introduced by Guido van Rossum during 1985- 1990. Python is an easy to learn, powerful programming language. Python is a general-purpose interpreted, interactive, object-oriented, and high-level programming languagewith dynamic semantics. Python is one of those rare languages which can claim to be both simple and powerful. The Python source code is available under the GNU General Public License (GPL). 

Python's elegant syntax and dynamic typing, together with its interpreted nature, make it an ideal language for scripting and rapid application development in many areas on most platforms.Python supports modules and packages, which encourages program modularity and code reuse. The Python interpreter and the extensive standard library are accessible in source or binary form without charge for all key platforms and can be freely distributed. 

Learning Python does not need any pre- requisites. However one should have the basic understanding of computer programming languages. A basic understanding of any of the programming languages is a plus.
